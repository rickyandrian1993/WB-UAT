=== Driver Installation ===

	1. Java JDK - jdk-17_windows-x64_bin
			- Open folder WB Preparement > Installer > Etc
			- Double click jdk-17_windows-x64_bin

	2. DB Postgresql - postgresql-14.5-1-windows-x64
			- Open folder WB Preparement > Installer > Etc
			- Double click postgresql-14.5-1-windows-x64
			- username: postgres
			- password: postgres
			- port: 5432

	3. NFC
			- Open folder WB Preparement > Installer > NFC > ACR122U SDK > SDK
			- Double click setup
			
	4. Fingerprint
			- Need Java JDK (If not installed yet)
			- Open folder WB Preparement > Installer > Fingerprint > UareUWin300.1115_2 > SDK > x64
			- Double click setup

			- Open folder WB Preparement > Installer > Fingerprint > wbf_driver > setup
			- Double click setup64

			- Open Device Manager >
				Biometric Devices > 
				U are U 4500 > 
				Tab Driver > 
				Update Driver > 
				Browse my computer for driver > 
				Let me pick from computer > 
				Pilih U are u 4500 Fingerprint Reader > 
				next



=== Setting Startup exe ===
This tutorial for make automaticly some service run on startup windows
	Open win + r
		-> shell:startup
		-> create file bat kemudian masukan code berikut
		-> start javaw -Xmx200m -jar "directory file jar"
	Contoh
		start javaw -Xmx200m -jar C:\WB Preparement\Services\External.jar
		start "" C:\WB\Setup\Timbangan.exe
